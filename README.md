# Civil Engineering Analysis – Project 3

## Project Overview
This project analyzes transportation-related datasets provided for a Federal Highway Administration (FHWA)-style client prompt. The goal of the project is to organize, visualize, and interpret data from the National Household Travel Survey (NHTS) and/or the Next Generation Simulation (NGSIM) dataset to support transportation planning decisions.

The project includes:
- A bar chart comparing categories within a selected variable
- A histogram showing the distribution of a selected variable
- A boxplot showing the distribution of a variable across categories
- At least two time-series plots related to driving behavior
- A simulation study using the Intelligent Driver Model (IDM)

## Purpose
The purpose of this project is to demonstrate how Python can be used to clean, analyze, visualize, and interpret transportation data in a way that is useful for engineering decision-making.

## User Guide
Follow these steps to run the notebook successfully:

1. Download or clone this repository.
2. Open the project folder in Jupyter Notebook or JupyterLab.
3. Make sure all files remain in their original folders, especially the files in the `data/` folder.
4. Open `Project3_Transportation_Analysis.ipynb`.
5. Run the cells from top to bottom in order.
6. If prompted to update file paths, make sure they point to the files inside the `data/` folder.
7. Review the generated tables, figures, and simulation outputs.

## Notes on Sequential Inputs
The notebook is intended to be run sequentially from the first cell to the last cell. Some later outputs depend on earlier data cleaning and variable selection steps. For that reason, cells should not be skipped.
